Реализовать форму входа на сайт в формате веб-страницы.

Окно входа должно иметь два поля:

Поле для ввода логина; требования к полю следующие:

допустимы только символы латиницы в нижнем регистре (a-z) и цифры (0-9). Пример: 0qwerty123456789

при вводе символов в верхнем регистре они должны сразу переводиться в нижний регистр. Пример: HellO WORLD -> hello world

Если с клавиатуры введён пробельный символ он должен игнорироваться

Поле для ввода пароля

должно скрывать любой текст звёздочками

Далее должна быть кнопка "Вход" и ссылка на страницу регистрации. Страницу регистрации можно не делать либо сделать пустой.

Оформление/дизайн страницы остаётся на усмотрение разработчика. Можно придумать самому, можно скопировать любой понравившийся вариант в интернете.
